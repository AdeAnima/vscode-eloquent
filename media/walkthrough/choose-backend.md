# Choose Your TTS Engine

Eloquent supports multiple text-to-speech backends:

| Backend | Best for | Size |
|---------|----------|------|
| **Kokoro** ⭐ | Fast, high-quality, no Python needed | ~80 MB |
| **F5-TTS** | Voice cloning on Apple Silicon | ~1.5 GB |
| **Custom** | Your own HTTP TTS server | — |

**Kokoro** is recommended for most users — it runs entirely in Node.js with no extra dependencies.

Click the button below to choose your backend:

[Choose TTS Backend](command:eloquent.setup)
